# 矩阵与线性变换

线性变换可以理解为就是对向量的一种变化，一个向量经过某种函数，变成另一个向量的这么一个关系。

如输入向量X[1,1,1]输出向量y[3,3,3]这么一个意思。

而矩阵其实相当于这其中变换的一个媒介，充当着怎么变换的这么一个意思。



## 矩阵

我们知道向量描述的是一种模型，而基向量代表的是可以描绘这些向量的在这个空间的一个基本单位，任意的向量都是可以通过基向量通过线性组合进行表示的。

而这个空间，一个向量V，通过某种变换变成了另一个向量V2，其实对于V和V2来说，他们变换前和变换后和原来基向量的线性组合的关系都是保持一致的。

可以理解为：

```
V = a * i + b * j
```

那么变换后

```
v2 = a * i (变) + b * j (变)
```

他们还是基于基向量i乘a加上基向量j乘b相加的这么一个结果，维度变的只有基向量。

所以任意向量通过某种变换变成某个变量，可以看成在这个空间里，他的基向量跟着这个空间一起变换了，连带着自己也变换了。

而这种变换的公式，其实就是 向量和矩阵相乘。
