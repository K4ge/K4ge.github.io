# 总览

线性代数在计算机领域应用是很多的，比如说游戏、数学建模、人工智能等等。

- 游戏中人物的各种特征，皮肤、发色、心情、坐标等等各种维度都需要实时地在程序中进行建模体现，并投射到屏幕中。

- 人工智能领域如每个token所代表的特征的各种含义也需要通过向量进行表示。

  而线性代数的各种变化等为这些场景表示进行了抽象打包，能够更好地将这些事物抽象成可以由计算机表示的数学特征。

所以我对线性代数的理解，就是它是一种建模的工具，同时对抽象建模出来的数学表示，提供了一套完整的线性变化规则。比如缩放，相加等，能够由计算机进行更好地数据处理，完成各项工作。
