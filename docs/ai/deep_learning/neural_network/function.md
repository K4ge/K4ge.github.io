# 函数

我喜欢把深度学习的第一程选为函数。我们怎么去理解这个世界呢，其实这个世界本质就是一个函数，我们接受各种各样的输入，通过函数转换成各种各样的输出。那对于多复杂的模型，多少B的模型，本质还是函数，有输入通过各种神经网络，最终变成输出。神经网络本质其实就是一个函数。



## 最简单的函数

最简单的函数，就是这种线性函数，中学学过a是斜率，b是偏移量。就这玩意，这其实就是最简单的神经网络了，输入一维，输出一维。

```
y = ax + b
```

其实通过这个最简单的函数就可以理解很多一些概念了，比如训练，比如推理，前向传播，反向传播，loss等等。有些复杂的东西其实通过最简单的模型去看本质，其实更好理解。



当然这些通过这个模型来看这些概念相对都比较简单，就不扩展了~



## 线性回归

这个概念听到很多，怎么去理解线性回归这个概念了，其实和训练也挺类似的，就是我们有很多的数据去拟合一条直线规律，训练推出那么一个函数的过程，我理解就叫线性回归。

有点抽象，反正大概就是这么个过程~
